# Property Index Archive

This repo contains one `index.html` file from each property repo, renamed using:

`index(Property-Name-MM-DD-YY).html`

The date is the source repository's GitHub `created_at` date.

## Contents

- Files: `indexes/`
- Import manifest: `manifest.csv`
- Import script: `import_property_indexes.sh`
